Gulp4简单使用

发表于2020-12-26|更新于2020-12-26|部署与构建Gulp
|字数总计:1.9k|阅读时长:9分钟

简介

gulp 是前端自动化打包构建工具，所谓打包可以理解为把文件压缩, 整合, 移动, 混淆。gulp 是一种基于流的打包构建工具。

文档：https://www.gulpjs.com.cn/docs/api/concepts/

使用 gulp，首先要全局安装gulp，全局安装只是可以使用其命令

也可以局部安装，通过scripts运行。

常用 API

  1. task()

    创建一个基于流的任务

  2. src()

    找到源文件

  3. dest()

    把一个内容放入指定目录内

  4. watch()

    监控指定目录下的文件, 一旦发生变化, 从新执行后面的任务

  5. series()

    逐个执行多个任务, 前一个任务结束, 第二个任务开始

  6. parallel()

    并行开始多个任务

  7. pipe()

    管道函数，所有的 gulp API 都是基于流。接收当前流, 进入下一个流过程的管道函数

常用插件

  1. gulp-cssmin

    导入以后得到一个处理流文件的函数

    https://www.npmjs.com/package/gulp-cssmin

  2. gulp-autoprefixer

    导入以后得到一个处理流文件的函数，直接再管道函数里面使用, 需要传递参数

    https://www.npmjs.com/package/gulp-autoprefixer

  3. gulp-sass

    编译scss和sass文件为css文件

    导入后直接使用即可

  4. gulp-less

    与gulp-sass类似，编译less文件为css文件

  5. gulp-uglify

    压缩js文件

    此压缩不可以压缩ES6，因此如果有 ES6 代码，需要先转为ES5

  6. gulp-babel

    专门进行 ES6 转 ES5 的插件。

    gulp-babel 有两个版本，8 主要用于 gulp4 中，7 主要用于 gulp3 中。默认安装最新版。

  7. gulp-htmlmin

    压缩 html

  8. del

    删除文件目录

  9. gulp-connect

    启动一个服务器，用于自动构建。

  10. gulp-pug

    编译pug为html文件

  11. gulp-concat

    合并js/css文件
